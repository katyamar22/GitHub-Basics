1. A terminal is where you can type in commands on your computer for your operating system to run. Some commands that you can run are creating files, deleting files, installing software as well as running git commands.

2. Git is a version control system. The purpose of Git is to track changes that are made in code. GitHub is a website for these repositories.

3. Forking allows developers to copy the source code locally. You can fork a public repository by clicking the fork button at the top right of the page.
